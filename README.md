# Sophia – AI-Powered Desktop Voice Assistant

Sophia is an AI-powered desktop voice assistant developed as a Final Year Project for the Bachelor of Science in Computer Science at the University of Agriculture, Faisalabad.

The application allows users to control their computer using natural voice commands. It can open desktop applications, launch websites, play YouTube videos, and respond with voice feedback while providing an interactive graphical user interface.

The system is built using Python for backend processing, Eel for connecting the Python backend with a modern web-based frontend, and SQLite for storing customizable voice commands. Sophia is designed to provide fast, secure, and locally executed automation without relying heavily on cloud services.

## Features

- Voice-controlled desktop automation
- Speech-to-text command recognition
- Text-to-speech voice responses
- Open desktop applications using voice commands
- Launch websites using voice commands
- Play YouTube videos through voice input
- SQLite database for customizable commands
- Interactive GUI with real-time status updates
- Modular architecture for easy future enhancements
